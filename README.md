# ribbon-helper

This repository contains a small React app used as the GitHub Pages site for "Ribbon Helper".

Quick commands:

- Install dependencies:

```bash
npm install
```

- Run development server:

```bash
npm run dev
```

- Build for production:

```bash
npm run build
```

- Deploy to GitHub Pages (uses `gh-pages`):

```bash
npm run deploy
```

The site title is "Ribbon Helper". It currently provides a header with three selections: `Switch Compatibility`, `Berry blending`, and `Contest move planning`. Click any selection to see a placeholder "coming soon" message.
