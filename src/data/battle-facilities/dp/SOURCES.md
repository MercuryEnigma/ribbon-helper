# Diamond/Pearl Battle Tower Sources

- Bulbapedia supplies canonical trainer names/classes and the listed nature for each Pokemon set.
- Altissimo supplies set labels, trainer rosters, battle progression, Palmer teams, and the code-backed trainer IV tiers.
- The 950 ordinary Pokemon sets are matched between both sources by species, item, moves, and EV distribution. EV values of 252 and 255 are treated as stat-equivalent only for that comparison.
- Bulbapedia's trainer summary currently lists different IV tiers. The generated data intentionally uses Altissimo's code-backed `3/6/9/12/15/18/21/31` tiers.
- Diamond/Pearl replaces the listed set natures with random natures in actual battles. The listed nature is retained as the calculator default.
