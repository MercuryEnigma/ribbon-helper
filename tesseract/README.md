# Local Tesseract assets

These files allow the HOME screenshot analyzer to run without a runtime CDN:

- `worker.min.js` is from `tesseract.js` 6.0.1.
- The SIMD and non-SIMD LSTM core files are from `tesseract.js-core` 6.1.2.
- `eng.traineddata.gz` is the English model distributed with Tesseract 5.5.2.
